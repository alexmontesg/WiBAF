Conditional Inclusion of Fragments

AHA! lets you conditionally include or hide parts of a page. The condition is an expression using attributes of (user model) concepts and constants. The condition is included in an if tag. One or two block tags are used for the conditional fragments. They correspond to a "then" and "else" construct in programming languages. An example: 

<if expr="tutorial.menus.show=1">
<block>
This part is included when in application "tutorial"
the attribute "show" of concept "menus" has the value 1.
</block>
<block>
This optional second part is included when the first part
is not included.
</block>
</if>

Conditions can contain Boolean combinations of expressions, like: 

"tutorial.pages.knowledge&gt;50 and tutorial.fragments.knowledge&lt;30"
It would have been nice to be able to write < and > instead of &lt; and &gt; but unfortunately this would confuse the XML parser.
If strings are used in expressions they must be put in single quotes ('), so as to not confuse the quotes with the double quotes (") around the whole expression.
